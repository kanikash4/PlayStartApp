# hello-playFramework

## Template created using sbt ##

 1. To create the template : 
     ```sh
    sbt new playframework/play-java-seed.g8
    ```

## Steps to run on locathost:  ##

 1. Run the service : 
     ```sh
    sbt run
    ```
And then go to http://localhost:9000 to see the running web application.
